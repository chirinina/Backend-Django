Filtro de categorías por nombre
-------------------------------

Ruta de la vista: inventario/views.py (función categorias)

Lógica principal:

    def categorias(request):
        # Captura de query params (ej.: ?nombre=PALABRA_FILTRAR)
        filtros = {
            "nombre": (request.GET.get('nombre') or "").strip()
        }

        categorias_qs = Categoria.objects.all().order_by('nombre')

        # Aplica filtro por nombre de manera insensible a mayúsculas
        if filtros["nombre"]:
            categorias_qs = categorias_qs.filter(nombre__icontains=filtros["nombre"])

        contexto = {
            "categorias": categorias_qs,
            "filtros": filtros,
            "hay_filtro": bool(filtros["nombre"]),
            "total_filtrado": categorias_qs.count(),
            "total_general": Categoria.objects.count(),
        }

        return render(request, 'categorias.html', contexto)

Puntos clave:
- Usa request.GET.get('nombre') para leer el parámetro de búsqueda.
- El filtro es insensible a mayúsculas (lookup icontains) y elimina espacios con strip().
- Ordena las categorías por nombre para una lectura consistente.
- El template recibe totales general y filtrado para mostrarlos en la UI.
- El enlace para limpiar el filtro mantiene intactas las demás vistas del proyecto.
